Cryptography

Cryptography is the science of securing information by transforming it into unreadable formats. It ensures confidentiality, integrity, authenticity, and non-repudiation of data during storage and transmission.

1. Core Objectives

  • Protect data from unauthorized access
  • Ensure data integrity and authenticity
  • Enable secure communication over untrusted networks

2. Types of Cryptography

  • Symmetric Encryption: Same key for encryption and decryption
  • Asymmetric Encryption: Public and private key pair
  • Hashing: One-way transformation for data integrity
  • Digital Signatures: Verify authenticity and integrity

3. Common Algorithms

  • AES (Advanced Encryption Standard)
  • RSA (Rivest–Shamir–Adleman)
  • SHA-256 (Secure Hash Algorithm)
  • Elliptic Curve Cryptography (ECC)

4. Example: Symmetric Encryption

Plaintext: "Hello World"

Key: "mysecretkey"

Encrypted: "A1B2C3D4..."

Decrypted: "Hello World"

5. Example: Hashing

Input: "password123"

SHA-256 Output: "ef92b778bafe771e89245b89ecbc08a44..."

Note: Hashing is irreversible and used for verification

6. Cryptographic Protocols

  • SSL/TLS (Secure communication)
  • PGP (Email encryption)
  • IPsec (Network layer security)

7. Key Management

  • Secure generation and storage of keys
  • Key rotation and expiration policies
  • Use of hardware security modules (HSM)

8. Best Practices

  • Use strong, modern algorithms
  • Never hardcode keys in source code
  • Encrypt sensitive data at rest and in transit
  • Verify integrity using hashes and signatures

9. Challenges

  • Key leakage or theft
  • Weak or outdated algorithms
  • Performance overhead in real-time systems

10. Applications

  • Secure messaging and email
  • Digital identity and authentication
  • Blockchain and cryptocurrency
  • Secure file storage and sharing
