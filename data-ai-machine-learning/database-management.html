Database Management Systems (DBMS)

A Database Management System is software that enables users to define, create, maintain, and control access to databases. It ensures data integrity, security, and efficient retrieval.

1. Goals of DBMS

  • Data Storage: Organize and persist data
  • Data Retrieval: Efficient querying and access
  • Data Integrity: Ensure accuracy and consistency
  • Security: Control access and protect data

2. Components of DBMS

  • Database Engine: Core service for data storage and retrieval
  • Query Processor: Parses and executes queries
  • Transaction Manager: Ensures ACID properties
  • Storage Manager: Manages physical data storage

3. Types of DBMS

  • Relational DBMS (RDBMS): Structured data in tables (e.g., MySQL, PostgreSQL)
  • NoSQL DBMS: Unstructured or semi-structured data (e.g., MongoDB, Cassandra)
  • Hierarchical DBMS: Tree-like structure (e.g., IBM IMS)
  • Network DBMS: Graph-like relationships (e.g., Integrated Data Store)

4. Relational Model Example

Table: Students
+----+--------+------+
| ID | Name   | Age  |
+----+--------+------+
| 1  | Alice  | 20   |
| 2  | Bob    | 22   |

5. SQL Overview

  • DDL (Data Definition Language): CREATE, ALTER, DROP
  • DML (Data Manipulation Language): SELECT, INSERT, UPDATE, DELETE
  • DCL (Data Control Language): GRANT, REVOKE

6. SQL Example

CREATE TABLE Students (
  ID INT PRIMARY KEY,
  Name VARCHAR(50),
  Age INT
);

INSERT INTO Students VALUES (1, 'Alice', 20);
SELECT * FROM Students WHERE Age > 21;
