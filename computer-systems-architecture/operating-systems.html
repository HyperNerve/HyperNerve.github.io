Operating Systems

An Operating System (OS) is system software that manages computer hardware, software resources, and provides common services for computer programs. It acts as an intermediary between users and the hardware.

1. Functions of an Operating System

  • Process Management: Scheduling, creation, termination of processes
  • Memory Management: Allocation, paging, segmentation
  • File System Management: Organization, access, permissions
  • Device Management: Drivers, I/O coordination
  • User Interface: CLI or GUI for user interaction
  • Security & Access Control: Authentication, authorization

2. Types of Operating Systems

  • Batch OS: Executes batches of jobs without user interaction
  • Time-Sharing OS: Allows multiple users to share system resources
  • Real-Time OS: Guarantees response within strict time constraints
  • Distributed OS: Manages multiple systems as a single unit
  • Embedded OS: Lightweight OS for embedded devices
  • Network OS: Provides services over a network

3. Process and Thread

  • Process: An executing instance of a program
  • Thread: A lightweight unit of execution within a process
  • Multitasking: Running multiple processes concurrently
  • Multithreading: Multiple threads within a single process

4. Memory Management Techniques

  • Paging: Divides memory into fixed-size pages
  • Segmentation: Divides memory into variable-sized segments
  • Virtual Memory: Uses disk space to simulate additional RAM

5. File Systems

  • Hierarchical Structure: Directories and subdirectories
  • File Attributes: Name, type, size, permissions
  • Access Methods: Sequential, direct, indexed
  • Examples: FAT32, NTFS, ext4, XFS

6. Scheduling Algorithms

  • FCFS: First Come First Serve
  • SJF: Shortest Job First
  • Round Robin: Time-sliced scheduling
  • Priority Scheduling: Based on process priority
  • Multilevel Queue: Multiple queues for different types of processes

7. I/O Management

  • Polling: CPU checks device status periodically
  • Interrupts: Devices signal CPU when ready
  • DMA: Direct Memory Access for fast data transfer

8. Security and Protection

  • Authentication: Verifying user identity
  • Authorization: Granting access rights
  • Encryption: Protecting data confidentiality
  • Access Control: File and resource permissions

9. Examples of Operating Systems

  • Windows: GUI-based OS by Microsoft
  • Linux: Open-source, Unix-like OS
  • macOS: Unix-based OS by Apple
  • Android: Linux-based OS for mobile devices
  • RTOS: VxWorks, FreeRTOS for embedded systems
