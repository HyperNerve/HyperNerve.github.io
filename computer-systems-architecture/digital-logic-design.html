Digital Logic Design

Digital Logic Design is the study of electronic circuits that operate on discrete signals. It forms the basis of all digital computing systems, enabling arithmetic, control, memory, and communication functions through binary logic.

1. Number Systems

  • Binary (Base-2): Digits: 0, 1. Used internally by all digital systems.
  • Octal (Base-8): Digits: 0–7. Compact representation of binary (3 bits per digit).
  • Decimal (Base-10): Digits: 0–9. Human-readable format.
  • Hexadecimal (Base-16): Digits: 0–9, A–F. Compact binary representation (4 bits per digit).
  • Conversions: Binary ↔ Decimal ↔ Hex ↔ Octal using positional weights.

2. Boolean Algebra

  • Variables: Represent binary values (0 or 1).
  • Operators: AND (·), OR (+), NOT (¬), XOR (⊕), NAND, NOR.
  • Identities:
    • Commutative: A + B = B + A
    • Associative: (A + B) + C = A + (B + C)
    • Distributive: A · (B + C) = A·B + A·C
    • De Morgan’s Laws: ¬(A·B) = ¬A + ¬B
  • Simplification: Reduces gate count and improves circuit speed.

3. Logic Gates

Gate Symbol Function Truth Table
AND · 1 if both inputs are 1 00→0, 01→0, 10→0, 11→1
OR + 1 if any input is 1 00→0, 01→1, 10→1, 11→1
NOT ¬ Inverts input 0→1, 1→0
XOR ⊕ 1 if inputs differ 00→0, 01→1, 10→1, 11→0

4. Combinational Logic Circuits

  • Definition: Output depends only on current inputs.
  • Half Adder: Adds two bits. Outputs: Sum, Carry.
  • Full Adder: Adds two bits and carry-in. Outputs: Sum, Carry-out.
  • Multiplexer (MUX): Selects one input from many.
  • Demultiplexer (DEMUX): Routes input to one of many outputs.
  • Encoder: Converts active input line to binary code.
  • Decoder: Converts binary input to active output line.

5. Sequential Logic Circuits

  • Definition: Output depends on current inputs and previous states.
  • Flip-Flops: Memory elements that store 1 bit.
    • SR Flip-Flop: Set/Reset
    • D Flip-Flop: Data latch
    • JK Flip-Flop: Toggle behavior
    • T Flip-Flop: Toggle on clock
  • Registers: Group of flip-flops to store multi-bit values.
  • Counters: Sequential circuits that increment/decrement values.
  • FSM (Finite State Machine): Control logic with defined states and transitions.

6. Karnaugh Maps (K-Maps)

  • Purpose: Visual method to simplify Boolean expressions.
  • Grouping: Combine adjacent 1s in powers of 2 (1, 2, 4, 8...)
  • Minimization: Reduces gate count and improves performance.

7. Timing and Clocking

  • Clock Signal: Synchronizes sequential operations.
  • Setup Time: Minimum time before clock edge that input must be stable.
  • Hold Time: Minimum time after clock edge that input must remain stable.
  • Propagation Delay: Time for signal to travel through a gate or circuit.

8. Programmable Logic Devices

  • PLA: Programmable Logic Array with configurable AND/OR planes.
  • PAL: Programmable Array Logic with fixed OR plane.
  • FPGA: Field-Programmable Gate Array with reconfigurable logic blocks and routing.
  • CPLD: Complex Programmable Logic Device, optimized for control logic.

9. Logic Families

  • TTL (Transistor-Transistor Logic): Fast, widely used in early computers.
  • CMOS (Complementary Metal-Oxide Semiconductor): Low power, used in modern ICs.

10. Real-World Applications

  • CPU Design: ALU, control unit, register file
  • Embedded Systems: Digital controllers, signal processors
  • Communication Systems: Encoders, decoders, modulators
  • Digital Signal Processing: Filters, FFT units
  • Robotics: Sensor interfacing, motor control logic
