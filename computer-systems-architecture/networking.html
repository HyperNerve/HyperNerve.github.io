Networking and Communication

Networking and communication systems enable data exchange between computing devices over wired or wireless media. They form the backbone of distributed computing, internet services, and modern digital infrastructure.

1. Goals of Networking

  • Connectivity: Enable devices to communicate
  • Reliability: Ensure accurate and complete data transfer
  • Scalability: Support growing number of devices and users
  • Security: Protect data and resources from unauthorized access

2. Network Types

  • LAN (Local Area Network): Covers small geographic area
  • WAN (Wide Area Network): Covers large geographic area
  • MAN (Metropolitan Area Network): Spans a city or campus
  • PAN (Personal Area Network): Short-range communication (e.g., Bluetooth)

3. Network Topologies

  • Bus: Single communication line
  • Star: Central hub connects all nodes
  • Ring: Nodes connected in a circular fashion
  • Mesh: Every node connects to every other

4. OSI Model (7 Layers)

  • Physical: Transmission of raw bits
  • Data Link: Frame creation, MAC addressing
  • Network: Routing, IP addressing
  • Transport: Reliable delivery (TCP, UDP)
  • Session: Manage sessions between applications
  • Presentation: Data translation, encryption
  • Application: End-user services (HTTP, FTP)

5. TCP/IP Protocol Suite

  • Application Layer: HTTP, SMTP, DNS
  • Transport Layer: TCP (reliable), UDP (fast)
  • Internet Layer: IP, ICMP
  • Network Access Layer: Ethernet, Wi-Fi

6. Communication Protocols

  • HTTP/HTTPS: Web communication
  • FTP/SFTP: File transfer
  • SMTP/IMAP: Email services
  • DNS: Domain name resolution

7. Addressing and Routing

  • IP Addressing: IPv4 and IPv6
  • MAC Addressing: Unique hardware identifiers
  • Routing Protocols: RIP, OSPF, BGP

8. Wireless Communication

  • Wi-Fi: IEEE 802.11 standards
  • Bluetooth: Short-range PAN
  • Cellular: 4G, 5G networks
  • Zigbee: Low-power mesh networks

9. Network Security

  • Firewalls: Control incoming/outgoing traffic
  • VPNs: Secure remote access
  • Encryption: Protect data in transit
  • Authentication: Verify user/device identity

10. Applications of Networking

  • Internet Access: Browsing, streaming, communication
  • Cloud Services: Remote storage and computing
  • IoT: Device interconnectivity
  • Telemedicine: Remote diagnostics and monitoring
