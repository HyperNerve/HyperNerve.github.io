Logic Programming

Logic programming is a declarative paradigm where programs are expressed in terms of facts and rules. Computation is performed by querying these facts using logical inference.

1. Core Concepts

  • Facts: Basic assertions about objects or relationships
  • Rules: Logical implications that define relationships
  • Queries: Questions posed to the system to infer answers
  • Inference Engine: Resolves queries using facts and rules

2. Example: Family Relationships in Prolog

Facts

parent(john, mary).
parent(mary, susan).
female(mary).
female(susan).

Rules

mother(X, Y) :- parent(X, Y), female(X).
grandparent(X, Y) :- parent(X, Z), parent(Z, Y).

Query

?- mother(mary, susan).
true.

?- grandparent(john, susan).
true.

3. Language Features

  • Unification: Matching variables and values during inference
  • Backtracking: System explores multiple paths to find solutions
  • Recursion: Rules can call themselves to express complex logic

4. Popular Logic Programming Languages

  • Prolog: Most widely used logic programming language
  • Datalog: Subset of Prolog used in databases
  • Mercury: Strongly typed logic-functional hybrid

5. Applications

  • Expert Systems: Encode domain knowledge as rules
  • Natural Language Processing: Grammar and parsing
  • Knowledge Representation: Ontologies and semantic reasoning
  • AI Planning: Goal-driven action sequences

6. Advanced Example: Path Finding

Facts

connected(a, b).
connected(b, c).
connected(c, d).

Rules

path(X, Y) :- connected(X, Y).
path(X, Y) :- connected(X, Z), path(Z, Y).

Query

?- path(a, d).
true.

7. Declarative Nature

Unlike imperative programming, logic programming focuses on what to solve, not how to solve it. The inference engine handles execution.

8. Limitations

  • Performance: Slower than imperative approaches for some tasks
  • Debugging: Can be difficult due to implicit control flow
  • Scalability: Complex logic may require optimization

9. Integration with Other Paradigms

  • Constraint Logic Programming: Combines logic with constraint solving
  • Answer Set Programming: Declarative problem solving with stable models

10. Tools and Environments

  • SWI-Prolog: Open-source Prolog implementation
  • GNU Prolog: Native compiler for Prolog
  • TuProlog: Lightweight Prolog for Java integration
