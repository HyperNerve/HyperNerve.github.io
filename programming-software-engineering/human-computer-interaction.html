Human-Computer Interaction (HCI)

Human-Computer Interaction is the study and design of user interfaces and experiences that enable effective interaction between people and digital systems. It combines computer science, psychology, design, and ergonomics.

1. Goals of HCI

  • Usability: Make systems easy and efficient to use
  • Accessibility: Ensure inclusivity for all users
  • User Satisfaction: Create pleasant and productive experiences

2. Components of HCI

  • User: Human interacting with the system
  • Interface: Medium through which interaction occurs
  • Task: Goal the user wants to accomplish

3. Interaction Styles

  • Command Line: Text-based input (e.g., Bash)
  • Menu-Based: Select options from lists
  • Direct Manipulation: Drag-and-drop, sliders
  • Form Fill-In: Structured data entry
  • Conversational: Chatbots and voice assistants

4. Usability Principles

  • Learnability: Easy to learn for new users
  • Efficiency: Fast task completion
  • Memorability: Easy to remember after a break
  • Error Tolerance: Prevent and recover from mistakes
  • Satisfaction: Pleasant to use

5. Example: Login Interface Evaluation

Criteria:
  - Clear labels for username and password
  - Feedback for incorrect login
  - "Forgot Password" option
  - Accessible via keyboard and screen reader

6. User-Centered Design Process

  • Understand Users: Research needs and behaviors
  • Design Prototypes: Create wireframes and mockups
  • Evaluate: Test usability and gather feedback
  • Iterate: Refine based on results

7. Accessibility Guidelines

  • WCAG: Web Content Accessibility Guidelines
  • Keyboard Navigation: Support non-mouse users
  • Screen Reader Compatibility: Use semantic HTML
  • Color Contrast: Ensure readability for all

8. Cognitive Models

  • GOMS: Goals, Operators, Methods, Selection rules
  • Fitts’s Law: Predicts time to target UI elements
  • Hick’s Law: Decision time increases with choices

9. Evaluation Techniques

  • Heuristic Evaluation: Experts review against usability principles
  • User Testing: Observe real users performing tasks
  • Surveys and Interviews: Gather subjective feedback

10. Tools for HCI Design

  • Figma: Collaborative UI design
  • Adobe XD: Prototyping and wireframing
  • Axure: Advanced interaction modeling
  • UsabilityHub: Remote user testing
