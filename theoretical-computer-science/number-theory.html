Number Theory

Number Theory is the study of integers and their properties. In computer science, it underpins cryptography, hashing, modular arithmetic, and algorithmic number manipulation.

1. Divisibility and Primes

  • Divides: a divides b if b = a × k for some integer k
  • Prime Number: Integer > 1 with no divisors other than 1 and itself
  • Composite Number: Integer with additional divisors

2. Greatest Common Divisor (GCD)

  • Definition: Largest integer that divides both a and b
  • Euclidean Algorithm: Efficient method to compute GCD

Example:

gcd(48, 18) → gcd(18, 12) → gcd(12, 6) → gcd(6, 0) = 6

3. Least Common Multiple (LCM)

Smallest positive integer divisible by both a and b

lcm(a, b) = (a × b) / gcd(a, b)

4. Modular Arithmetic

  • Definition: a ≡ b (mod m) if m divides (a − b)
  • Addition, Multiplication, Exponentiation: All defined modulo m

Example:

7 + 5 ≡ 0 (mod 12)

5. Congruences

  • Linear Congruence: ax ≡ b (mod m)
  • Solution: Exists if gcd(a, m) divides b

6. Modular Inverse

Integer x such that a × x ≡ 1 (mod m)

  • Exists if gcd(a, m) = 1
  • Computed using Extended Euclidean Algorithm

7. Fermat’s Little Theorem

If p is prime and a is not divisible by p:

a^(p−1) ≡ 1 (mod p)

8. Euler’s Theorem

If a and n are coprime:

a^φ(n) ≡ 1 (mod n)

Where φ(n) is Euler’s totient function: number of integers ≤ n that are coprime to n

9. Chinese Remainder Theorem (CRT)

Solves systems of congruences with pairwise coprime moduli:

x ≡ a₁ (mod m₁)
x ≡ a₂ (mod m₂)
...

Has a unique solution modulo M = m₁ × m₂ × ...

10. Prime Factorization

  • Every integer > 1 can be uniquely factored into primes
  • Used in RSA encryption

11. Fast Exponentiation

Computes a^b mod m efficiently using binary exponentiation

12. Applications

  • Cryptography: RSA, Diffie-Hellman, ECC
  • Hashing: Prime moduli for uniform distribution
  • Algorithms: Modular arithmetic in number-theoretic algorithms
  • Error Detection: Checksums and cyclic redundancy checks
