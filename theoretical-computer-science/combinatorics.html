Combinatorics

Combinatorics is the study of counting, arrangement, and selection of discrete objects. It provides tools for analyzing algorithms, data structures, and probabilistic models in computer science.

1. Basic Counting Principles

  • Sum Rule: If task A can be done in m ways and task B in n ways, and they are mutually exclusive, total = m + n
  • Product Rule: If task A can be done in m ways and task B in n ways, total = m × n

2. Permutations

  • Definition: Ordered arrangements of objects
  • Formula: P(n, r) = n! / (n − r)!
  • Example: Ways to arrange 3 out of 5 books

3. Combinations

  • Definition: Unordered selections of objects
  • Formula: C(n, r) = n! / (r!(n − r)!)
  • Example: Ways to choose 3 fruits from 5

4. Binomial Theorem

Expands powers of binomial expressions:

(a + b)ⁿ = Σ (n choose k) aⁿ⁻ᵏ bᵏ

5. Pigeonhole Principle

If n items are placed into m containers and n > m, then at least one container has more than one item.

Example:

In a group of 13 people, at least two share a birth month.

6. Inclusion-Exclusion Principle

Used to count elements in the union of overlapping sets:

|A ∪ B| = |A| + |B| − |A ∩ B|

General Form:

|A₁ ∪ A₂ ∪ ... ∪ Aₙ| = Σ|Aᵢ| − Σ|Aᵢ ∩ Aⱼ| + Σ|Aᵢ ∩ Aⱼ ∩ Aₖ| − ...

7. Combinatorial Proofs

Prove identities by interpreting both sides as counting the same set in different ways.

Example:

C(n, r) = C(n − 1, r − 1) + C(n − 1, r)

Choose r items from n: either include a specific item or not.

8. Recurrence Relations

  • Definition: Equations that define sequences recursively
  • Example: Fibonacci: F(n) = F(n − 1) + F(n − 2)
  • Solving: Use iteration, characteristic equations, or generating functions

9. Generating Functions

Formal power series used to encode sequences and solve recurrence relations.

Example:

G(x) = a₀ + a₁x + a₂x² + ...

10. Catalan Numbers

  • Definition: Count certain recursive structures
  • Formula: Cₙ = (1 / (n + 1)) × C(2n, n)
  • Applications: Parenthesizations, binary trees, Dyck paths

11. Partitions and Compositions

  • Partition: Ways to write n as a sum of positive integers (order doesn't matter)
  • Composition: Ordered partitions of n

12. Applications

  • Algorithm Analysis: Counting operations, recursion depth
  • Cryptography: Combinatorial designs and key generation
  • Probability: Sample space enumeration
  • Data Structures: Tree shapes, heap configurations
